# angular6-admin-lte

* Admin-lte implementation for Angular 6
* Node version v10.10.0
* npm version 6.4.1

## Installation

```bash
$ git clone https://github.com/eligomes/angular6-admin-lte.git
$ cd angular6-admin-lte
$ npm i
```

## Run demo
```bash
$ ng serve
```

* admin-lte-css
